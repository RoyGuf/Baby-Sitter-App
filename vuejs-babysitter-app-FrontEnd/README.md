
# Babysitter-Service-App - Front End

The Client side of the **Babaysitter-Service-App**, runs on port 3000 by default. Built with Vue JS 3 framework.

## Installation

#### Environment Variables

To run this project, you will need to add the following environment variables to your ```.env``` file

| Keys | Description | Default value
| ------ | ------ | ------ |
| `VITE_BACKEND_URL` | The back-end API url | http://localhost:8080/api |

#### Build

Build a ```./dist``` folder which then can be copy and served on the server-side.

```bash
  npm run build
```

#### Run Locally

Clone the project

```bash
  git clone https://github.com/RoyGuf/Baby-Sitter-App.git
```

Go to the project directory

```bash
  cd vuejs-babysitter-app-FrontEnd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
